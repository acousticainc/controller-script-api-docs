Easy to Use

The Mixcraft JavaScript API makes writing control surface scripts easy and flexible. An integrated script editor with Logging and a MIDI monitor allow you to work right in Mixcraft if you want to.

MIDI and SysEx data for quick and universal use

The JavaScript API allows for control surfaces that send and recieve MIDI and SysEx data.
